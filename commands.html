All Commands

User Commands: The following commands work for all users.

  • %help - Displays all commands. (Usually more up-to-date than the website)
  • %help admin - Displays all admin commands. (Usually more up-to-date than the website)
  • %secret - Responds with a secret. Currently 300+
  • %pickup - Responds with a random pickup line from a hand-curated list of over 100.
  • %invite - Invite Barbara to your Discord Server!
  • %link - Displays a customizable message when the command is run.

Admin Commands. The following commands only work for server owners and people they allow (and the JCWYT team ðŸ˜‰).
  • %admin list roles - Lists roles that can use admin commands.
  • %admin list users - Lists users that can use admin commands.
  • %admin add role [@role] - Adds role to admin command whitelist.
  • %admin add user [@user] - Adds user to admin command whitelist.
  • %admin remove role [@role] - Removes role from admin command whitelist.
  • %admin remove user [@user] - Removes user from admin command whitelist.

    • Config commands:

  • %link set [message or link] - Customizes the link in %link. Add as many as you want.
  • %prefix [new prefix] - Customize the prefix for all commands.
  • %config read - Reads the config as a JSON object. Currently, there aren't commands for many config values, so you'll have so look here to see them all.
  • %config set [parameter] [value] - Set a parameter in config to the value you stated.
