User Commands

The following commands work for all users.

%help - Displays all commands.

%secret - Responds with a secret.

%pickup - Responds with a pickup line.

%link - Displays a customizable link, and a link to the JCWYT website.

Admin Commands

The following commands only work for server owners and people they allow.

%admin list roles - Lists roles that can use admin commands.

%admin list users - Lists users that can use admin commands.

%admin add role [@role] - Adds role to admin command whitelist.

%admin add user [@user] - Adds user to admin command whitelist.

%admin remove role [@role] - Removes role from admin command whitelist.

%admin remove user [@user] - Removes user from admin command whitelist.

%link custom [message or link] - Customizes the link in %link. Add as many as you want.

%prefix [new prefix] - Customize the prefix for all commands from % to whatever you want.
