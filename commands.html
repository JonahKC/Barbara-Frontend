User Commands

The following commands work for all users.

%help - Displays all commands.

%help admin - Displays all admin commands. (See below)

%secret - Responds with a secret. There's currently 300+ secrets, all made by JCWYT!

%pickup - Responds with a random pickup line from a hand-curated list of over 100.

%fact - Returns a random fact, but can only be run once every 10 minutes. This command uses an API, see below for credit links.

%dadjoke - Returns a random dad joke, runnable once every 10 minutes. This command uses an API, see below for credit links.

%say [message] - Sends a message as Barbara. If you're not an admin, it mentions who the message is by.

%say button [buttonURL] [buttonTitle] [message] - Sends a message as Barbara with a button at the bottom that takes you to a URL. If you're not an admin, it mentions who the message is by.

%link - Displays a customizable message when the command is run.

The following commands only work for server owners and people they allow (and the JCWYT team ðŸ˜‰).

Admin Role Commands

%admin list roles - Lists roles that can use admin commands.

%admin list users - Lists users that can use admin commands.

%admin add role [@role] - Adds role to admin command whitelist.

%admin add user [@user] - Adds user to admin command whitelist.

%admin remove role [@role] - Removes role from admin command whitelist.

%admin remove user [@user] - Removes user from admin command whitelist.

Server Customization

%link set [message or link] - Customizes the link in %link. Add as many as you want.

%prefix [new prefix] - Customize the prefix for all commands.

Advanced Server Customization

%config read - Reads the config as a JSON object. Currently, there aren't commands for many config values, so you'll have so look here to see them all.

%config set [parameter] [value] - Set a parameter in config to the value you stated.
